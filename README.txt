-- Características

- Sistema simples de gestão de colaboradores, estilo CRUD


-- Execução
Para executar a aplicação, por favor, siga os seguintes passos:
	1. Após descompactar a pasta zipada no diretório de sua escolha, abra o projeto na IDE desejada;
	2. Execute a classe GestaoColaboradoresApplication;
	3. Digite no navegador que preferir o seguinte endereço: localhost:8080
	4. Pronto! Basta navegar pela aplicação.

P.S.: para acessar os benefícios atribuídos a cada colaborador, por favor, clique no link associado a cada funcionário cadastrado.
